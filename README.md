# Customer Churn Prediction

ML classification model to predict customer churn with 4 algorithms, ROC curves, and feature importance analysis.

## Models Compared
- Logistic Regression
- Random Forest
- Gradient Boosting
- SVM (Support Vector Machine)

## Key Features Used
- Contract type, Tenure, Monthly fee
- Support calls, Satisfaction score
- Internet service, Payment method
- Engineered: high_support, long_tenure

## Tech Stack
`Python` `Scikit-learn` `Pandas` `Matplotlib` `Seaborn`

## Setup
```bash
git clone https://github.com/gunnu1106/customer-churn-prediction.git
cd customer-churn-prediction
pip install -r requirements.txt
python model.py
```
